Sharepic generator

Welcome to the Rheinos-and-RheinIIs-Social-Media-Sharepic-Generator-Overview-Page

I couldn't make this word any longer. Here's an overview over out templates :)

Standard Post

Template for everything that hasn't a template. Use this for the post after a (fantasy) tournament or after a training camp or everything else. If you use it for the post after a non-fantasy tournament, you can put score posts behind this in the post!

Score Post

Template to post the result of a game after a game. Please don't make a post for every single group stage game! You can either put all results from group stage in a single post with multiple pictures or just put the result in the story after a game.

Score Story

Template to post the result of a game after a game in the story.

Schedule Post

Template to announce league gamedays and other events with a schedule (e.g. tournament group stages). If there are more than five games scheduled, split the schedule equally on multiple pictures, which should be posted in a single post. You can leave out games without one of our teams! But please don't use this template to announce a single game (e.g. a knockout stage game)! In this case you can use the announcement template instead!

Announcement Post

Template for all kinds of announcements (except open trainings). Whenever there are great news or just normal news, use this template, e.g. when we're about to play a game in knockout stage.

If you use this template to post about a game (e.g. to say that we're playing the quarter finals vs Team X at Y o'clock in the livestream), please use this as announcement only (prior to game) and use the score template for results afterwards. Also please don't announce every single group stage game :)

Roster Post

Template to post our rosters, should be self-explaining.

Medal Post

Template for when we've reached a top 3 placement on a bigger tournament, e.g. ODSC isn't big. For lower placements, use the standard template and put the placement in the caption.

Open Training Announcement Post

Template to announce open trainings. Please use it only once per open training and pin the post in our instagram profile.

Open Training Info Picture

Put this template at the end of recruitment posts. It should contain details about the open training.

Message Post

Template to spread a message we want to spread. Use this for things like the PSD Vereinspreis or the first picture on a informational recruitment posts (e.g. "Quadball is mixed Gender").

Informational Post

Use this template for informational content, e.g. to explain the rules of Quadball. In most cases it makes sense to use a message template picture for the first picture of the post, then multiple informational pictures and at the end an open training info picture.
